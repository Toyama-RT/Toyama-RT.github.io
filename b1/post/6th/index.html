このサイトについてをつくる

このサイトについてをメニューに表示させるには、

[[Menus.main]] Name = “このサイトについて” URL = “/about/” weight = 3

をconfig.tomlに追記して、

hugo new about/about.md

で、ページを作ればできる。

同じやり方で、書いているやつはこんなだ。とかいろいろできる。
